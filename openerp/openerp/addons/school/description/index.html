Openerp School Module

The OpenERP-school allows you to store education establishments to manage student data.

It helps schools to manage admission registers, student profiles and reminders.

Install the “school”

First we will install the “school” from the addons
Click on Settings – Modules – Modules
Search the module name in the Name Field in the search view of modules After searching by name you will get the icon and after clicking on “Install” button you can install this module.

After installing this module you will get three menus.

EMS – Admission Register – Admission Registers
EMS – Students Profile – Students Profiles
EMS – Reminders – Reminder

There are four groups in this module.If you select these groups you can see configuration menu.

1)School Administrator
2)Teachers
3)Students
4)Parents

Configuration Menu:

EMS – Configuration – School Config :
It will allow you to create school related information.
EMS – Configuration – Academic Years :
It will allow you to create academic years for school, ex.2011,2012,2013.
EMS – Configuration – Standards :
It will allow you to create standards for school, ex. 10th, 11th, 12th.
EMS – Configuration – Subject :
It will allow you to create subjects for school, ex. Maths,English,Science,Gujarati etc.

Admission Registers

Admission register will allow you to enter all the details of students like personal information,school information,class information,family details, previous school information and remarks if any.
Student's profile is created after his/her admission is confirm and to confirm the student admission you can click on the “Done” button.
Events, Award Info, Medical, and Ceratificates tabs are added after student is in “Done” state. You can also print your registration form.

Student Profiles

Student profiles have all the information of students.It is same as admission register.

Reminders

Reminders are used to remind the school related activities and events.
